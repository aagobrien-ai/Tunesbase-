# TuneBase Version 1 — Agreed Workflow Review

This review build separates the three example tunes by purpose:

- Cooley's Reel: master tune-page and Learn Tune template with main staff notation and chords.
- Saddle the Pony: the single guided sound-recognition test.
- Mug of Brown Ale: duplicate/existing-record check before a tune is added.

The homepage now presents TuneBase as a growing 250+ tune platform rather than featuring these three tunes as public front-page content. The included seed database currently contains 206 records; notation, sound fingerprints and verification remain to be completed across the catalogue.

Deploy the folder to Netlify. ABCJS loads from its CDN on first use.
